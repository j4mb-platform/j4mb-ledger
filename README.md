# j4mb-ledger
Event-driven double-entry ledger service built with Spring Boot and PostgreSQL.

--------

# Overview

The ledger service is the financial core of the J4MB platform.

It is responsible for:
- account management
- journal entries
- transaction recording
- balance calculation
- financial auditability

The service is designed with:
- clean architecture
- domain-oriented design
- transactional consistency
- future event-driven extensibility

----

# Technology Stack

- Java 21
- Spring Boot 3
- PostgreSQL
- Docker Compose
- Spring Data JPA
